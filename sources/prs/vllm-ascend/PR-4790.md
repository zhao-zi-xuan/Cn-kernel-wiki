## Summary

### What this PR does / why we need it?

This PR adapt DispatchGmmCombineDecode operator to parameters of small operators.
1. This operator no longer requires permuting the weights and scales of GMM1.
2. This operator no longer requires transposing the weights of GMM2.

Therefore, this operator and the small operator can use the same parameters (weights and scales), which is beneficial for m
